# Nuit-de-l-info-2025
L'idée principale de notre projet est de faire une mini-aventure interactive où l'on va pouvoir répondre à des questions et, selon notre score, réussir ou non notre aventure.
Les questions seront sur le thème de la démarche NIRD.
Les défis choisis et implémentés :
  - Sopra steria - L'ergonomie : simplifier pour mieux vivre. : Consiste à un pseudo à écrire dans un container, sauf que l'écriture ne se passe pas comme prévu...
  - Viveris - Chat'bruti : Consiste à un Chatbot qui répond à côté de la plaque.
  - Autocut - Hidden snake : Un serpent qui se balade sur l'écran et qui lorsque l'on clique dessus lance un snake.
  - IUT Marne-La-Vallée - En trois dimensions : Intégration d'élément 3D dans notre site. Implémentation de la 3D avec le snake. Le snake se baladera sur un cube 3D.
  - Capgemini - Visualisation Audio : Un visuel pour l'audio. Implémentation musique en fond, et suivit son en arrière plan lors du mini-jeu snake.
